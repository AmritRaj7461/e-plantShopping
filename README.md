# 🌿 Paradise Nursery - Shopping Cart Application

## 📌 Project Description
Paradise Nursery is a React-based e-commerce web application where users can browse plants, view details, and manage a shopping cart. The application demonstrates dynamic UI updates using Redux and modern frontend development practices.

---

## 🚀 Features
- 🌱 Browse plants by categories (Indoor, Outdoor, Succulent)
- 🖼️ View plant details (image, name, price)
- 🛒 Add to Cart functionality
- 🔄 Increase / Decrease item quantity
- ❌ Remove items from cart
- 💰 Dynamic total price calculation
- 📊 Cart count updates in navbar
- 🔗 Navigation using React Router

---

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- React Router DOM
- CSS
